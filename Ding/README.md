# 定额估算系统 - 演示版

## 🎯 项目简介
这是一个基于 FastAPI 和 HTML/JavaScript 的定额估算系统演示版本，模仿广联达定额软件的功能设计，提供项目管理、定额模板、估算计算和报表导出功能。

## 🚀 快速启动

### 1. 启动后端服务

```bash
# 进入后端目录
cd D:\001\Ding\backend

# 启动 FastAPI 服务器
python -m uvicorn app.main:app --port 8000 --host 127.0.0.1 --reload
```

后端服务将在 `http://127.0.0.1:8000` 启动

### 2. 打开前端界面

在浏览器中直接打开：
```
D:\001\Ding\frontend\index.html
```

## 📋 功能说明

### 项目管理
- 查看所有项目
- 创建新项目（设置项目名称、币种、税率）
- 演示数据已预置一个示例项目

### 定额模板
- 查看所有定额模板
- 创建新模板（包含章节和条目）
- 演示数据已预置一个通用模板

### 估算功能
- 选择项目和模板
- 加载模板条目
- 输入工程量
- 自动计算：
  - 小计 = Σ(数量 × 单价)
  - 税费 = 小计 × 税率
  - 总计 = 小计 + 税费

### 报表导出
- 查看估算明细
- 导出为 CSV 文件（包含明细和汇总）

## 🔧 API 接口文档

启动后端后访问：
- Swagger UI: http://127.0.0.1:8000/docs
- ReDoc: http://127.0.0.1:8000/redoc

### 主要接口

#### 项目接口
- `POST /projects` - 创建项目
- `GET /projects` - 获取所有项目

#### 模板接口
- `POST /templates` - 创建模板
- `GET /templates` - 获取所有模板
- `GET /templates/{template_id}` - 获取指定模板

#### 估算接口
- `POST /estimates` - 创建估算
- `GET /estimates` - 获取所有估算
- `GET /estimates/{estimation_id}` - 获取指定估算
- `GET /estimates/{estimation_id}/export.csv` - 导出估算为CSV

## 📊 数据结构

### 项目 (Project)
- id: 项目ID
- name: 项目名称
- currency: 币种（如 CNY）
- tax_rate: 税率（如 0.13 表示 13%）

### 模板 (Template)
- id: 模板ID
- name: 模板名称
- version: 版本号
- sections: 章节/分组列表
  - code: 章节编码
  - name: 章节名称
  - items: 条目列表
    - code: 条目编码
    - name: 条目名称
    - unit: 单位
    - unit_price: 单价

### 估算 (Estimation)
- id: 估算ID
- project_id: 关联的项目ID
- template_id: 关联的模板ID
- lines: 明细行列表
- subtotal: 小计金额
- tax: 税费金额
- total: 总金额

## 📝 使用示例

### 创建估算

1. 在前端界面中选择项目（如 "示例项目 (P-001)"）
2. 选择模板（如 "通用模板 (1.0)"）
3. 点击"加载模板条目"按钮
4. 在各条目中输入工程量
5. 点击"生成估算"按钮
6. 查看计算结果并可选择导出CSV

### 导出CSV

点击估算结果中的"导出CSV"按钮，将下载包含明细和汇总的CSV文件。

## 🚨 注意事项

1. **数据存储**：当前版本使用内存存储，重启后端后数据会丢失
2. **浏览器兼容性**：推荐使用现代浏览器（Chrome、Firefox、Edge）
3. **端口占用**：确保 8000 端口未被占用
4. **中文编码**：CSV 导出可能需要使用支持 UTF-8 的编辑器打开

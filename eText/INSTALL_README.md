# E文件解析工具 - 安装包说明

## 文件清单

本目录包含E文件解析工具的完整安装包：

### 应用程序文件
- `dist/EFileParser/` - 目录版本（推荐）
  - `EFileParser.exe` - 主程序（约1.7MB）
  - `_internal/` - 依赖文件目录
- `dist/EFileParser_Single.exe` - 单文件版本（约35MB）

### 安装脚本
- `install.bat` - Windows安装程序
- `uninstall.bat` - Windows卸载程序

### 源代码文件
- `qt_app.py` - Qt6主应用程序
- `efile_parser.py` - E文件解析器

### 示例文件
- `20260114/` - 示例E文件目录

## 安装方法

### 方法1：使用安装程序（推荐）

1. 以管理员身份运行 `install.bat`
2. 按提示输入安装目录（或按回车使用默认目录）
3. 等待安装完成
4. 双击桌面上的"E文件解析工具"快捷方式启动

### 方法2：直接运行

1. 进入 `dist/EFileParser/` 目录
2. 双击 `EFileParser.exe` 运行
3. 或运行单文件版本 `dist/EFileParser_Single.exe`

### 方法3：复制到任意位置

1. 复制整个 `dist/EFileParser/` 文件夹到任意位置
2. 双击 `EFileParser.exe` 运行

## 使用说明

### 启动程序
- 双击桌面快捷方式或主程序 `EFileParser.exe`

### 解析E文件
1. 点击"选择文件"按钮
2. 选择 .dat 格式的E文件
3. 点击"解析"按钮
4. 查看解析结果

### 查看数据
- **文件概览**标签页：查看文件基本信息和统计
- **详细数据**标签页：选择数据段查看详细表格数据
- **原始内容**标签页：查看文件的原始文本内容

### 导出报告
1. 解析完成后，点击"导出报告"按钮
2. 选择保存位置
3. 报告将以 .txt 格式保存

## 卸载方法

### 方法1：使用卸载程序
1. 以管理员身份运行 `uninstall.bat`
2. 按提示输入安装目录
3. 确认卸载

### 方法2：手动删除
1. 删除安装目录（默认：`C:\Program Files\EFileParser`）
2. 删除桌面快捷方式

## 技术支持

### 常见问题

**Q: 程序无法启动？**
A: 请确保您的系统是64位Windows 10或11。如果仍有问题，请以管理员身份运行。

**Q: 解析失败？**
A: 请确认：
- 文件格式正确（.dat）
- 文件编码为UTF-8
- 文件未损坏

**Q: 程序运行缓慢？**
A: 大文件解析可能需要几秒到几分钟，请耐心等待。

**Q: 如何更新程序？**
A: 下载新版本，运行 install.bat 覆盖安装即可。

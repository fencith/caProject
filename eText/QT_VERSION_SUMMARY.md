## 三、功能实现

### ✅ 核心功能

#### 3.1 文件上传
- 支持点击选择文件
- 支持拖拽文件
- 自动识别.dat格式

#### 3.2 E文件解析
- 解析所有6类E文件
  - FDJZ（风机/光伏单机信息）
  - SYZXX（升压站信息）
  - ZTXX-QXHJ（总体气象信息）
  - DQYC（短期功率预测）
  - CDQYC（超短期功率预测）
  - TJXX（统计信息）
- 自动提取文件头、数据段、数据表
- 支持UTF-8编码

#### 3.3 数据展示
- **文件概览**标签页
  - 文件基本信息
  - 统计信息（数据段数、数据表数、数据行数）

- **详细数据**标签页
  - 数据段列表
  - 完整数据表格
  - 支持选择不同数据段

- **原始内容**标签页
  - 显示文件原始文本
  - 等宽字体便于阅读

#### 3.4 报告导出
- 导出为.txt格式
- 包含完整的解析结果
- 结构化输出

## 四、安装方法

### 快速安装

```bash
# 方法1：使用安装程序（推荐）
以管理员身份运行 install.bat

# 方法2：直接运行
进入 dist/EFileParser/ 目录
双击 EFileParser.exe
```

### 安装位置
- 默认：`C:\Program Files\EFileParser`
- 可自定义安装目录

### 桌面快捷方式
安装后自动创建桌面快捷方式："E文件解析工具"

---

## 五、使用说明

### 启动程序
1. 双击桌面快捷方式
2. 或直接运行 `EFileParser.exe`

### 解析文件
1. 点击"选择文件"按钮
2. 选择 .dat 格式的E文件
3. 点击"解析"按钮
4. 等待解析完成（通常1-5秒）

### 查看数据
- 切换不同标签页查看不同视图
- 在"详细数据"标签页点击数据段查看对应数据

### 导出报告
1. 解析完成后点击"导出报告"
2. 选择保存位置
3. 报告自动生成

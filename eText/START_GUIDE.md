# 国家电投昆明生产运营中心E文件解析工具 - 快速启动指南

## 🚀 5分钟快速上手

### 步骤1：启动程序

```bash
# 方法1：双击运行
dist/EFileParser_SPIC/EFileParser_SPIC.exe

# 方法2：使用安装程序
以管理员身份运行 install.bat
```

### 步骤2：使用工具

1. **查看界面** - 左上角显示国家电投LOGO
2. **选择文件** - 点击"选择文件"按钮
3. **解析文件** - 点击"解析"按钮
4. **查看结果** - 切换标签页查看数据
5. **导出报告** - 点击"导出报告"保存
6. **查看规范** - 点击"下载规范文件"打开PDF

## ❓ 常见问题

**Q: 程序无法启动？**
A: 以管理员身份运行

**Q: LOGO不显示？**
A: 已内置在程序中，请重新安装

**Q: PDF无法打开？**
A: 程序已内置PDF，点击"下载规范文件"即可

**Q: 解析失败？**
A: 确认文件是.dat格式，编码为UTF-8

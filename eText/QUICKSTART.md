# E文件解析工具 - 快速启动指南

## 🚀 快速启动

### 方法1：使用批处理文件（推荐）

双击 `start.bat` 文件即可启动应用

### 方法2：手动启动

```bash
python app.py
```

应用将在 http://localhost:5000 启动

## 📁 项目文件说明

```
eText/
├── app.py                    # Flask主应用
├── efile_parser.py          # E文件解析器
├── requirements.txt          # Python依赖包
├── start.bat                 # Windows启动脚本
├── templates/
│   └── index.html           # 上传页面
├── uploads/                 # 上传文件和生成报告存储目录
└── 20260114/               # 示例E文件目录
```

## 📖 使用说明

1. **启动应用**
   - 双击 `start.bat` 或运行 `python app.py`
   - 浏览器访问 http://localhost:5000

2. **上传文件**
   - 点击上传区域选择 .dat 文件
   - 或直接拖拽 .dat 文件到上传区域

3. **解析文件**
   - 点击"开始解析"按钮
   - 等待解析完成（通常几秒）

4. **查看报告**
   - 点击"查看完整报告"链接
   - 报告将在新标签页中打开
   - 可以打印或保存为PDF

## 🔧 故障排除

### 端口被占用
如果5000端口被占用，修改 `app.py` 最后一行：
```python
app.run(debug=True, port=5001)  # 改为其他端口
```

### 依赖包问题
重新安装依赖：
```bash
pip install -r requirements.txt
```

### 文件上传失败
- 确认文件为 .dat 格式
- 文件大小不超过16MB
- 文件编码为UTF-8

## 📝 报告内容

生成的HTML报告包含：

1. **文件概览**
   - 数据段数量
   - 数据表数量
   - 数据行数
   - 文件头信息

2. **详细数据**
   - 每个数据段的基本信息
   - 所有数据表和表格数据
   - 完整的数据行展示

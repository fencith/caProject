## 🚀 使用方法

### 方法1：直接运行
```bash
# 双击运行
dist/EFileParser_SPIC_Final.exe
```

### 方法2：安装使用
```bash
# 以管理员身份运行
install.bat

# 安装后双击桌面快捷方式
```

### 使用步骤
1. 双击启动程序
2. 查看"国家电投昆明生产运营中心"LOGO
3. 点击"选择文件"选择.dat文件
4. 点击"解析"按钮
5. 查看解析结果（三个标签页）
6. 可选择"导出报告"保存

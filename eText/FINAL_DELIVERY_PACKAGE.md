## 🚀 安装使用指南

### 快速安装
```bash
# 1. 以管理员身份运行安装程序
install.bat

# 2. 按提示选择安装目录 (默认: C:\Program Files\SPIC_EFileParser)

# 3. 安装完成后，双击桌面快捷方式启动
```

### 直接运行
```bash
# 无需安装，直接双击
dist/EFileParser_SPIC_v1.1_fixed.exe
```

### 使用步骤
1. **启动程序** - 看到国家电投LOGO和完整标题
2. **选择文件** - 点击"选择文件"或拖拽E文件
3. **解析文件** - 点击"解析"按钮，自动检测编码
4. **查看结果** - 三个标签页显示不同视图
5. **导出报告** - 点击"导出报告"保存分析结果

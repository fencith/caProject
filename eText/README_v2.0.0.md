## 功能说明

### 文件选择
- 支持 .dat, .tar.gz, .tgz 文件
- 文件过滤器: "支持的文件 (*.dat *.tar.gz *.tgz)"

### 解析功能
- 多编码支持 (UTF-8, GBK, GB2312, GB18030, UTF-16)
- tar.gz 自动解压和合并解析
- 后台线程解析，不阻塞UI
- 进度条显示解析进度

### 数据显示
- 文件概览标签页
- 详细数据标签页
- 原始内容标签页
- 表格支持行列显示
- 数据段可点击查看详情

### 导出功能
- 导出报告到文本文件
- 包含完整解析结果
- 包含版本、版权、作者信息

### 数据存储
- SQLite 数据库自动保存解析结果
- 支持历史记录查询

### 主题系统
- 自动跟随 Windows 主题
- 浅色模式: 白色背景，黑色文字
- 深色模式: 深色背景，浅色文字
- 所有控件样式完整适配

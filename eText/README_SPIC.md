> 版本：1.0 | 版权：国家电投昆明生产运营中心 版权所有

## 简介

根据《南方电网新能源数据上送规范 V3.3修订版（2022.2月）》及总调水新〔2025〕28号文件，由国家电投昆明生产运营中心开发的E文件解析桌面应用。

## 快速开始

### 安装使用

```bash
# Windows用户
1. 以管理员身份运行 install.bat
2. 双击桌面快捷方式"国家电投E文件解析工具"
```

### 直接运行

```bash
# 进入目录版本
dist/EFileParser_SPIC/EFileParser_SPIC.exe
```

## 功能特点

✅ **国家电投品牌** - 集成国家电投LOGO和品牌标识
✅ **专业界面** - 采用国家电投绿（#006633）作为主色调
✅ **6类E文件支持** - FDJZ、SYZXX、ZTXX-QXHJ、DQYC、CDQYC、TJXX
✅ **实时解析** - 快速解析E文件数据
✅ **多标签页展示** - 文件概览、详细数据、原始内容
✅ **报告导出** - 支持导出为文本报告
✅ **规范文档** - 内置南方电网规范PDF下载功能
✅ **无需安装Python** - 已打包为独立EXE文件

## 界面说明

### 主界面
- **左上角**：国家电投LOGO（红绿色标识）
- **标题栏**：国家电投昆明生产运营中心E文件解析工具 v1.0
- **操作按钮**：选择文件、解析、导出报告、下载规范文件
- **状态栏**：显示版权信息

### 功能按钮
| 按钮 | 功能 |
|------|------|
| 选择文件 | 选择.dat格式的E文件 |
| 解析 | 开始解析文件 |
| 导出报告 | 保存为文本报告 |
| 下载规范文件 | 打开南方电网规范PDF |

### 数据展示标签页
- **文件概览**：文件基本信息和统计信息
- **详细数据**：选择数据段查看详细表格数据
- **原始内容**：查看文件的原始文本内容

## 支持的E文件类型

| 类型 | 说明 | 示例文件名 |
|------|------|-----------|
| FDJZ | 风机/光伏单机信息 | FD_GD.GDZJYHHB_FDJZ_*.dat |
| SYZXX | 升压站信息 | FD_GD.GDZJYHHB_SYZXX_*.dat |
| ZTXX-QXHJ | 总体信息和气象环境 | FD_GD.GDZJYHHB_ZTXX-QXHJ_*.dat |
| DQYC | 短期功率预测 | FD_GD.GDZJYHHB_DQYC_*.dat |
| CDQYC | 超短期功率预测 | FD_GD.GDZJYHHB_CDQYC_*.dat |
| TJXX | 统计信息 | FD_GD.GDZJYHHB_TJXX_*.dat |

## 系统要求

- 操作系统：Windows 10/11 (64位)
- 内存：至少4GB
- 磁盘空间：100MB

## 使用说明

### 启动程序
1. 双击桌面快捷方式"国家电投E文件解析工具"
2. 或直接运行 `EFileParser_SPIC.exe`

### 解析文件
1. 点击"选择文件"按钮
2. 选择 .dat 格式的E文件
3. 点击"解析"按钮
4. 等待解析完成（通常1-5秒）

### 查看数据
- 切换不同标签页查看不同视图
- 在"详细数据"标签页点击数据段查看对应数据

### 导出报告
1. 解析完成后点击"导出报告"
2. 选择保存位置
3. 报告自动生成（包含版权信息）

### 下载规范文件
1. 点击"下载规范文件"按钮
2. 自动打开PDF文件
3. 查看南方电网新能源数据上送规范

## 技术支持

### 常见问题

**Q: 程序无法启动？**
A: 请确保您的系统是64位Windows 10或11。如果仍有问题，请以管理员身份运行。

**Q: LOGO显示不正常？**
A: 程序内置了国家电投LOGO（spic_logo.png），如果显示异常，请重新安装。

**Q: 解析失败？**
A: 请确认：
- 文件格式正确（.dat）
- 文件编码为UTF-8
- 文件未损坏

**Q: 规范PDF无法打开？**
A: 程序内置了PDF文件，如果无法打开，请检查：
- PDF文件是否存在
- 是否有PDF阅读器

**Q: 程序运行缓慢？**
A: 大文件解析可能需要几秒到几分钟，请耐心等待。

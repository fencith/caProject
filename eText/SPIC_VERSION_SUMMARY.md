## 📝 使用说明

### 安装
```bash
# 以管理员身份运行
install.bat

# 安装到默认目录：C:\Program Files\SPIC_EFileParser
```

### 使用
1. 双击桌面快捷方式"国家电投E文件解析工具"
2. 查看界面上的国家电投LOGO和标题
3. 点击"选择文件"选择.dat文件
4. 点击"解析"进行解析
5. 点击"下载规范文件"查看PDF

### 导出报告
1. 解析完成后点击"导出报告"
2. 保存为.txt文件
3. 报告包含国家电投版权信息

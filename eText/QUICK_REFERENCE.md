# E文件解析工具 - 快速参考卡

## 🚀 5分钟快速上手

### 第一步：安装
```bash
# 以管理员身份运行
install.bat

# 或直接运行
dist/EFileParser/EFileParser.exe
```

### 第二步：使用
1. 点击"选择文件"按钮
2. 选择 .dat 文件
3. 点击"解析"
4. 查看结果

### 第三步：导出（可选）
点击"导出报告"保存为文本文件

## 🔧 故障排除

| 问题 | 解决方案 |
|------|---------|
| 程序无法启动 | 以管理员身份运行 |
| 解析失败 | 确认文件是.dat格式 |
| 显示乱码 | 确认文件编码为UTF-8 |
| 导出失败 | 检查写入权限 |
